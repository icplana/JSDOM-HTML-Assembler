Instructions

When you are finished execute in the browser console the function validateExercise14() to check your solution.

Add a click event listener for the Add list button that executes the addList function to:

  • create a ul list container with a class of list-container.
  • create as many li elements as week days in the days array using a loop. Each li child element should have a class of weekday-item and the text content of the day from the array.
  • the list should be added as a child element of the ex-wrapper div.
  • the list should only be added once.
  • HINT: you can use the childElementCount property to check if the parent has no child elements.
